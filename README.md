## TMD 인턴 2주차 업무: AI Challenge School Mobile App Prototype

Description:
“AI Challenge School”은 AI 도구(ChatGPT 등 6가지 AI 서비스)를 활용해 기본 개념을 학습하고, 퀴즈를 통해 이해도를 점검한 뒤, 최종적으로 성취(achievement)를 해금하는 구조의 모바일 앱 프로토타입입니다.
이 프로젝트는 수도권과 비수도권 간 디지털 리터러시 격차 문제를 해결하는 것을 목표로 설계되었습니다. 사용자가 AI 도구를 단순히 사용하는 것을 넘어, 학습 → 이해 → 성취의 과정을 경험하도록 구성되어 있습니다.

상세 기획 문서: https://docs.google.com/document/d/1AeJjk0vyv4TY0dL5sVKU4XTl2VIAnSV-wHc0cX03K8s/edit?tab=t.0#heading=h.6q0ymb3lowry. 

Key Features:
1. 6가지 AI 도구 기반 정보 학습 콘텐츠 제공
2. 체험형 학습 섹션 (Information / Experience)
3. 학습 내용을 확인하는 Quiz 시스템
4. 사용자 동기부여를 위한 Achievement 시스템

Development Notes:
1. 기존에 해커톤에서 개발했던 모바일 앱 프론트엔드 템플릿을 기반으로 확장
2. 정보 제공, 퀴즈, 업적 시스템 기능을 중심으로 구조 설계 및 구현
3. UI/UX 흐름을 “학습 → 평가 → 보상” 구조로 설계

Tech Stack:
1. Flutter
2. Dart
3. Android Studio
